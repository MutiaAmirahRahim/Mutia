1. kesimpulan pertemuan 1 Cara install wordpress.org 1. Buka alamat wordpress.org kemudian pilih tombol unduh wordoress.5.8.1 2. Buka file kemudian ekstrak 3. Rename folder yang telah di ekstrak (tanpaspasi) 4. Pindahkan folder xamps ke dalam htdocs 5. Buka browser kemudian ketikkan URL : http://localhost/nmfolderdihtdocs/wp-admin 6. Pilih bahasa tekan tombol Continue 7. Menyiapkan database, Klik ayo. 8. Mengisi Database dengan klik di mysQl admin. 9. Buat database 10. isi koneksi database name basis data : diisi dengan nama database Nama pengguna: diisi dengan root. Sandi : dikosongkan. Kemudian klik tombol Submit. 11. Masukkan detail mengenai wordpress, Nama situs : judul utama dari website Nama pengguna : username yang akan digunakan untuk login ke dashboard WordPress Sandi : password yang akan digunakan untuk login ke dashboard WordPress. Kemudian klik install wordpress 12. Jika proses instalasi sudah berhasil, login ke dashboard WordPress dengan cara klik tombol Login dan masukkan username dan password yang dibuat sebelumnya.
